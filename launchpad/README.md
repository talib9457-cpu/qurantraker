# 🚀 LaunchPad - لوحة التحكم السريعة

تطبيق Windows احترافي يجمع كل الأدوات والملفات والروابط والأوامر في **مكان واحد**.

---

## المميزات

| الميزة | التفاصيل |
|--------|----------|
| 📁 فتح مجلدات | استعراض أي مجلد بنقرة واحدة |
| 📄 فتح ملفات | فتح أي ملف بالبرنامج الافتراضي |
| 🌐 روابط ويب | فتح المواقع في المتصفح |
| ⚙️ تشغيل برامج | تشغيل أي تطبيق `.exe` |
| 💻 أوامر Windows | تشغيل PowerShell / CMD بصمت |
| 🔍 بحث سريع | بحث فوري في كل الأدوات |
| 🗂️ تصنيفات | تنظيم الأدوات في مجموعات |
| 🌙 Dark Mode | واجهة داكنة احترافية |
| 💾 حفظ تلقائي | البيانات تُحفظ في JSON محلياً |

---

## تشغيل التطبيق

```bash
# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python main.py
```

---

## بناء ملف EXE

```batch
# Windows
build.bat
```

```bash
# Linux / macOS
bash build.sh
```

الملف الناتج: `dist/LaunchPad.exe`

---

## اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl+N` | إضافة أداة جديدة |
| `Ctrl+F` | التركيز على البحث |
| `Escape` | مسح البحث |
| نقرة يمين | قائمة تعديل/حذف |

---

## مسار البيانات

```
C:\Users\[اسم المستخدم]\AppData\Local\LaunchPad\shortcuts.json
```

## التقنيات

- **Python 3.10+**
- **PyQt6** - واجهة المستخدم
- **JSON** - حفظ البيانات
- **PyInstaller** - تحويل إلى EXE
